# New Mac set up

In my current job as an IT Consultant for a large university I have found my self needing to configuring three different MacBooks for my daily use over the last three months. Trying to recall all the settings and apps that I use in my daily work flow became exhausting and so I decided to document my setup.

Everyone has different preferences, there is no right way to set up your MacBook, but when I was setting up my first MacBook for my current job I found that looking through someone else's setup and tech stack gave me new ideas, exposed me to new apps and workflows. So, my hope in creating this page is two fold:

1. I really want to speed up the process of setting up my work machine, and, to not have to search all of my notes trying to remember the name of that one app that did that specific thing.
2. Also, I hope it inspires you. Maybe you will find some new app that I have incorporated in to my workflow. Or, you will just reaffirm that there is no better way to set it up than the way you already have.

Cheers!

## OS Productivity

### Window Management

There are some excellent paid options out there, I have personally used [Magnet](https://magnet.crowdcafe.com/) for this in the past. Currently on my work machines I have been using [Rectangle](https://rectangleapp.com/), which is a free and open source app and it has been fantastic!

[**Rectangle**](https://rectangleapp.com/)

### Screen Shot App

If you dont mind paying [CleanshotX](https://cleanshot.com/) is an amazing option. I use screen shots to document a lot at work and also to share information, it is also import to me to be able to annotate my screen shots so I was ok paying some money for this app. My favorite feature is the "scrolling capture" which allows you to capture a whole webpage, or long string of text if needed.

[**CleanshotX**](https://cleanshot.com/)

[**Shottr**](https://shottr.cc/) is the most has been the most recommend free "screenshot" app. I have not used it yet, but when my new work machine arrives I will be trying this. 

Stay tuned for updates....

[**Shottr**](https://shottr.cc/) 

One last app that falls into the screen shot catagory is [**TextSniper**](https://textsniper.app/). No, it is not a video game. It is an OCR screen shot tool. What that means is it allows you to take a screen shot of a picture that has words in it, and it "automagically" coppies the text straight to your clip board. It is not free....but it has found a home in my daily workflow and was worth the investment.

[**TextSniper**](https://textsniper.app/)

### Notes app

This subject will get its own blog here one day as I love trying out new notes apps, but, for now lets review the options based on need.

My current daily driver is [**Obsidian**](https://obsidian.md/). This app can seem a little daunting right out of the gate, but if privacy, and data ownership are important to you...its worth it.

Obsidian is basically a fancy text editor that works with markdown(.md) files which make it incredablly fast. It does take some effort to get it set up for your workflow but the end result is a fast light weight knowledge managment tool that can run on any system.

My other choice is [**Craft**](https://www.craft.do/). [**Notion**](https://www.notion.so/) is to databases what [**Craft**](https://www.craft.do/) is to writing. This app is created to provide an amazing note taking experience and its capabilities are powerful. 
- I think about switching back at least once a day, so, I always keep it installed for that fateful day.
